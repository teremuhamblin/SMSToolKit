==========================
SMSC — Centre de service SMS
==========================

Le SMSC (Short Message Service Center) est l’élément central du fonctionnement des SMS.

Fonctions principales
=====================

- Réception des SMS envoyés par les téléphones
- Stockage temporaire
- Tentatives de livraison
- Gestion des échecs
- Application des règles d’encodage

Format du SMSC
==============

Le format standard est international :

    +33689004000

Le numéro varie selon l’opérateur.
