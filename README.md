# SMSToolKit
🧩 Description courte du projet SMSToolKit  SMSToolKit est un projet technique et pédagogique dédié au paramétrage, diagnostic et optimisation des SMS sur smartphones.   Il centralise les informations essentielles sur le fonctionnement des SMS, les paramètres opérateurs (SMSC), les formats d’encodage, les problèmes courants et leurs solutions.
