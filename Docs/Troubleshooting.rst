==========================
Dépannage des SMS
==========================

Cette section couvre les problèmes les plus courants liés aux SMS.

SMS qui ne partent pas
======================

- SMSC incorrect
- Problème SIM
- IMS/VoLTE désactivé
- Réseau saturé

SMS en retard
=============

- Congestion réseau
- TTL élevé
- Destinataire indisponible

SMS dupliqués
=============

- Réessais SMSC
- Mauvaise configuration IMS

Problèmes vers certains numéros
===============================

- Format incorrect
- Restrictions opérateur
