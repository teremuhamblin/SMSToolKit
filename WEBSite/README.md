🌐 WEBSite — README du projet

WEBSite est une base web professionnelle, simple, modulaire et évolutive, conçue pour servir de fondation à des projets personnels, associatifs ou de production.  
Le projet met l’accent sur une structure claire, une documentation propre et une architecture facile à maintenir.

---

📁 Structure du projet

```text
WEBSite/
├── src/                # Pages HTML, CSS, JS
├── assets/             # Images, icônes, médias
├── css/                # Styles globaux et modules
├── js/                 # Scripts et interactions
├── config/             # Fichiers de configuration
├── docs/               # Documentation du projet
└── tools/              # Scripts utilitaires
```

---

🎯 Objectifs du projet

- Fournir une base web professionnelle prête à l’emploi  
- Proposer une structure claire, modulaire et scalable  
- Faciliter l’intégration de nouvelles pages et fonctionnalités  
- Offrir une documentation simple et accessible  
- Servir de socle pour des projets plus avancés (SPA, design system, dashboard, etc.)

---

🧩 Contenu principal

src/
Contient toutes les pages HTML du site, organisées de manière cohérente et prêtes à être étendues.

css/
Styles globaux, modules, variables, thèmes.  
Pensé pour être harmonisé et facilement maintenable.

js/
Scripts de base, interactions simples, logique front-end.

assets/
Images, icônes, logos, médias.  
Structure optimisée pour GitHub Pages et projets web.

config/
Fichiers de configuration du projet (chemins, modules, paramètres).

docs/
Documentation technique, guides, notes de version, roadmap.

tools/
Scripts utilitaires pour automatiser certaines tâches (build, nettoyage, génération).

---

🚀 Utilisation

Aucune dépendance lourde :  
le projet fonctionne directement en ouvrant les fichiers HTML dans un navigateur.

Pour un workflow plus avancé :

- hébergement via GitHub Pages  
- intégration d’un serveur local (ex. live-server)  
- ajout d’un système de build si nécessaire (optionnel)
