# ⚡ Design System — SMSToolKit (CyberTech Edition)

Le design system définit l’identité visuelle, les composants, les couleurs et les règles UI/UX du projet.

---

## 🎨 Identité visuelle

### Style
- CyberTech
- Neon Blue
- Glow subtil
- Ombres douces
- Interfaces sombres

### Logo
- SVG animé
- Pulsation neon
- Compatible dark/light

---

## 🎨 Couleurs

| Nom | Valeur | Usage |
|-----|--------|--------|
| **Primary Neon** | #00C8FF | Actions, highlights |
| **Dark Base** | #0A0F14 | Background |
| **Dark Panel** | #111820 | Cards, modules |
| **Accent Purple** | #7A4DFF | Éléments interactifs |
| **Success** | #00FF9C | Validation |
| **Error** | #FF3B3B | Erreurs |

---

## 🅰️ Typographie

- **Font principale** : Inter  
- **Poids** : 300 / 400 / 600  
- **Taille** : 14–18px  

---

## 🧩 Composants

### **Buttons**
- arrondis légers
- glow neon
- hover animé
- variantes : primary, ghost, danger

### **Cards**
- fond sombre
- bordure neon subtile
- ombre douce

### **Inputs**
- bordure neon
- focus glow
- placeholder gris clair

---

## 📐 Spacing

- XS : 4px  
- S : 8px  
- M : 16px  
- L : 24px  
- XL : 32px  

---

## 🌗 Modes

### Dark Mode (par défaut)
- couleurs neon
- contrastes élevés

### Light Mode (optionnel)
- gris clair
- accents bleus

---

## 🧪 Règles UX

- feedback immédiat
- transitions douces
- animations < 200ms
- lisibilité prioritaire
