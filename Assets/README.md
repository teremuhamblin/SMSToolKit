# 🗂️ assets/

Le dossier **assets/** contient toutes les ressources visuelles utilisées par le projet **SMSToolKit**.  
Il centralise les éléments graphiques afin de garder une structure propre, organisée et facilement maintenable.

---

## 📁 Contenu du dossier

- **logo.svg** — Logo officiel du projet  
- **icons/** — Icônes techniques ou UI  
- **images/** — Illustrations, visuels, schémas  
- **animations/** — SVG animés, effets visuels  
- **banners/** — Bannières pour README ou site web  

---

## 🎯 Objectif

- Regrouper toutes les ressources graphiques au même endroit  
- Faciliter la maintenance et l’évolution du design  
- Assurer une cohérence visuelle sur l’ensemble du projet  
- Préparer les assets pour le site web
- Préférer les formats **SVG** pour les logos et icônes  
- Optimiser les images avant ajout (compression recommandée)  
- Organiser les fichiers dans les sous‑dossiers appropriés  
- Éviter les doublons et les fichiers inutilisés
