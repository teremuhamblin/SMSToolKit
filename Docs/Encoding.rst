==========================
Encodage des SMS
==========================

Deux encodages principaux sont utilisés dans le réseau GSM.

GSM 7-bit
=========

- Encodage historique
- 160 caractères
- Très compatible

UCS-2 (Unicode)
===============

- Utilisé pour les emojis, accents, caractères spéciaux
- 70 caractères
- Peut provoquer des SMS multiples

Impact de l’encodage
====================

- Longueur du message
- Nombre de segments
- Compatibilité réseau
